# Problem Set 0 Clarifications

- Number of bins:
  - There is no single correct answer
  - Pick something that communicates the shape well
- Matching distributions to data:
  - Focus first on the values a variable can take
  - Then ask which probability distributions support exactly those values
  - Then ask which parameters define those distributions
- Example:
  - A variable on $(-\infty, \infty)$ can often be modeled with a Gaussian
  - A Gaussian is defined by a location parameter $\mu$ and a scale parameter $\sigma^2$
