# Recap

<v-clicks depth="2">

- We can do inference with GLMs in many of the same ways we do with linear regressions
  - Some stuff is wonky!
- We can model count data using a variety of GLMs
  - Binomial for fixed numbers of trials
  - Poisson for unrestricted counts
  - Negative Binomial and Quasi-Poisson for overdispersed counts

</v-clicks>
