# Recap

- Naming clusters is similar to naming dimensions, factors, and topics
- Model-based clustering makes distributional assumptions about the data generating process
  - Allows for the use of model comparison metrics (like BIC)
  - Requires estimating a _lot_ of parameters!
- Mixture distributions and mixture models can be used to combine models when you think there might be multiple DGPs simultaneously!
